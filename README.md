# Embeddable Python IDE
Embedabble Python IDE that supports starter code in the URL.
Useful for educational pages.

## Features
* Allows starter code in URL
* Editing and running Python scripts
* Reset button to return to starter code

## Instructions
1. Access https://jcjeffreys.github.io/python-lesson-templates/index.html
2. Add your starter code
3. Copy the embed frame or URL and use it with your own site
