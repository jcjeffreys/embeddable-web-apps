# Embeddable web apps
A collection of web apps that you can embed in your web pages.
Intended for embedding in wiki pages for online Learning Management Systems such as Canvas.

## Python IDE
Embed a minimal Python IDE with code editor and output in your page.

### Features
* Allows starter code in URL
* Editing and running Python scripts
* Save and load .py files
* Change font size
* Reset button to return to starter code
* **Presentation mode** with larger, bold fonts for classroom projections

### Instructions
1. Access https://jcjeffreys.github.io/embeddable-web-apps/python-ide.html
2. Add your starter code
3. Use the buttons to copy the embed frame or URL

### Presentation Mode
For classroom presentations where code needs to be visible from the back of the room, add `&presentation` (or `&present`) to the URL:

**Normal mode:**
```
https://jcjeffreys.github.io/embeddable-web-apps/python-ide.html#code=BASE64CODE
```

**Presentation mode:**
```
https://jcjeffreys.github.io/embeddable-web-apps/python-ide.html#code=BASE64CODE&presentation
```

In presentation mode:
- Code editor font is enlarged to 24px and made bold
- Output terminal font is enlarged to 24px and made bold
- Font selector is hidden (presentation mode overrides it)
- Perfect for projecting on screens or large displays
